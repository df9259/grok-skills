# Grok Skills Repository

This repo hosts custom skills for Grok AI agents. Each skill is a self-contained module with SKILL.md describing its purpose, triggers, and detailed instructions.

## Structure
- `skill-name/SKILL.md`: Core skill definition
- Subdirs for references, scripts, templates, hooks as needed.

## Usage
Skills are loaded by Grok's skill system for enhanced capabilities in coding, architecture, prompt engineering, etc.

Clone and contribute via PRs or issues.
